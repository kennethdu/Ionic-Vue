# Ionic-Vue


## This is a basic remake of content from the first page of https://www.reddit.com/r/pics made through Ionic-Vue an app development platform and Axios a promise based HTTP client


# *Technologies Used* #

* Vue.js
* Javascript
* CSS
* Ionic-Vue
* Ionic-Core
* HTML
* Axios
* Vue CLI
